# 🎮 Video Game Sales Dashboard

An interactive **Power BI dashboard** analyzing global video game sales trends, including genre performance, platform distribution, regional sales, and publisher insights.

---

## 📊 Project Overview

This project focuses on transforming raw video game sales data into meaningful insights using **Power BI**.  
The dashboard provides a clear understanding of industry trends through interactive visualizations and data-driven analysis.

---

## 🎯 Objectives

- Analyze global video game sales data  
- Identify top-performing genres and platforms  
- Understand regional sales distribution  
- Visualize trends using an interactive dashboard  

---

## 📁 Dataset Information

The dataset contains historical sales data of video games across different regions and platforms.

### 🔹 Key Columns:
- Name (Game Title)  
- Platform  
- Year  
- Genre  
- Publisher  
- NA Sales  
- EU Sales  
- JP Sales  
- Global Sales  

---

## 📈 Dashboard Features

- 📌 KPI Cards (Total Sales, Average Sales, Total Games)  
- 📊 Sales by Genre  
- 🕹️ Sales by Platform  
- 📅 Year-wise Sales Trend  
- 🌍 Regional Sales Distribution  
- 🏢 Publisher Analysis  

---

## 🔍 Key Insights

- 🎮 Action genre dominates global sales  
- 🕹️ PlayStation platforms lead the market  
- 🌍 North America and Europe contribute the highest revenue  
- 📈 Sales peaked during the mid-2000s to early 2010s  
- 🏆 A few blockbuster games generate the majority of sales
